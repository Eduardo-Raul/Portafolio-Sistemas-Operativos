# Introducción a los sistemas operativos


Un sistema operativo es el software encargado de administrar los recursos de una computadora y permitir la interacción entre el usuario y el hardware

Su función principal es servir como intermediario entre las aplicaciones y los componentes fisicos del sistema, gestionando recursos como:

- procesador
- memoria
- dispositivos de entrada y salida
- archivos
- procesos


Sin un sistema operativo, cada programa tendría que controlar directamente el hardware, lo que haría mucho mas compleja la programación y el funcionamiento de las computadoras


Actualmente los sistemas operativos son utilizados en:

- computadoras personales
- servidores
- telefonos moviles
- dispositivos embebidos
- supercomputadoras


---


# Objetivos de un sistema operativo


Los sistemas operativos buscan cumplir varios objetivos importantes para el correcto funcionamiento de una computadora


Entre los objetivos principales se encuentran:

- facilitar el uso de la computadora
- administrar eficientemente los recursos
- permitir la ejecución concurrente de programas
- garantizar seguridad y estabilidad
- optimizar el rendimiento del sistema


El sistema operativo decide:

- qué proceso utiliza el CPU
- cómo se distribuye la memoria
- cuándo se accede a dispositivos
- cómo se almacenan los archivos


---


# Componentes principales de un sistema operativo


Un sistema operativo esta compuesto por varios componentes encargados de diferentes tareas


Algunos de los componentes mas importantes son:

- kernel
- administrador de procesos
- administrador de memoria
- sistema de archivos
- controladores de dispositivos
- interfaz de usuario


---


# Kernel


El kernel o nucleo es la parte central del sistema operativo y se encarga de comunicarse directamente con el hardware


Es el encargado de controlar:

- procesos
- memoria
- dispositivos
- interrupciones
- llamadas al sistema


Todas las aplicaciones dependen del kernel para poder utilizar los recursos del sistema


El kernel trabaja en modo privilegiado, teniendo acceso completo al hardware de la computadora


---


# Llamadas al sistema


Las llamadas al sistema son mecanismos que permiten que los programas soliciten servicios al sistema operativo


Por ejemplo:

- crear procesos
- abrir archivos
- leer datos
- escribir información
- reservar memoria


En sistemas UNIX y GNU/Linux las llamadas al sistema forman parte fundamental de la programación de sistemas


---


# Modos de operación


Los sistemas operativos trabajan principalmente en dos modos de ejecución:

- modo usuario
- modo kernel


El modo usuario limita el acceso directo al hardware para proteger el sistema

El modo kernel tiene acceso total a los recursos y ejecuta operaciones privilegiadas


Esto ayuda a mejorar la seguridad y estabilidad del sistema operativo


---



Los sistemas operativos han evolucionado constantemente desde las primeras computadoras hasta los sistemas modernos utilizados actualmente

En un principio, las computadoras no contaban con sistemas operativos y los programas se ejecutaban directamente sobre el hardware


Conforme aumentó la complejidad de los sistemas computacionales surgió la necesidad de administrar recursos de forma automatica y eficiente


Los sistemas operativos comenzaron a encargarse de:

- organizar tareas
- administrar memoria
- controlar dispositivos
- ejecutar multiples programas


Esto permitió mejorar el aprovechamiento del hardware y facilitar el trabajo de los usuarios y programadores


---



# Funciones principales de un sistema operativo


Entre las funciones mas importantes de un sistema operativo se encuentran:


## Administración de procesos


El sistema operativo controla la creación, ejecución y terminación de procesos

Tambien coordina el uso del procesador mediante algoritmos de planificación


---


## Administración de memoria


El sistema operativo decide cómo utilizar la memoria RAM y cómo asignarla a los procesos


Esto incluye:

- memoria virtual
- paginación
- segmentación


---


## Sistema de archivos


El sistema operativo organiza y administra la información almacenada en discos


Permite:

- crear archivos
- eliminar archivos
- leer datos
- escribir información
- administrar permisos


---


## Administración de dispositivos


El sistema operativo controla dispositivos de entrada y salida como:

- teclados
- discos
- impresoras
- tarjetas de red


Esto se realiza mediante controladores llamados drivers
